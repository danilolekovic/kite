### Information

## Features

  * Only half a kilobyte (`643b`)
  * Fast
  * Error handling
  * Multiple files

## Example

**index.html:**
```html
<kite src="test.html"></kite>
```

**test.html:**
```html
<p>Hello world!</p>
```

### Get It Now

Simply add this tag to the bottom of your HTML file.

```js
<script src="https://raw.githubusercontent.com/danilolekovic/kite/master/src/kite.min.js"></script>
```
